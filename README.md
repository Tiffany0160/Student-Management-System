# Student Management System

A simple Java console-based system for managing student records, including adding students, viewing them, updating marks, and calculating the average mark.

## Features
- Add student records (name and mark)
- View all students
- Update student marks
- Calculate average marks

## Technologies Used
- Java
- ArrayList
- Scanner class
- Basic OOP concepts

## How to Run
1. Open the project in a Java IDE
2. Run the `Student.java` file
3. Use the menu to manage student data
